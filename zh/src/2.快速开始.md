# 快速开始

#### 安装 
```
go get github.com/yu-org/yu
```  
#### 开发一个链上的 Writing 和 Reading  
```go
import (
	"github.com/yu-org/yu/apps/pow"
	. "github.com/yu-org/yu/core/types"
	"github.com/yu-org/yu/env"
	"github.com/yu-org/yu/common"
	"github.com/yu-org/yu/context"
	"github.com/yu-org/yu/startup"
	"github.com/yu-org/yu/tripod"
)

type Example struct {
	*tripod.Tripod
}

// 此处定制开发一个 Writing
func (e *Example) WriteA(ctx *context.WriteContext) error {
	caller := ctx.Caller
        // 设置该 writing 所需消耗的lei (lei和gas同义)
	ctx.SetLei(100)
        // 将数据存入链上状态中。
	e.Set(caller.Bytes(), []byte("yu"))
        // 向链外发射一个event
	ctx.EmitStringEvent("execute success")
	return nil
}

// 此处定制开发一个 Reading
func (e *Example) ReadA(ctx *context.ReadContext) error {
	caller := ctx.Caller
	value, err := e.Get(caller.Bytes())
	if err != nil {
		return err
        }
	ctx.String(string(value))
	return nil
}

```
 
#### 在main函数中添加 tripods

```go
func NewExample() *Example {
	df := tripod.NewTripod()
	e := &Example{df}

	// 此处需要手动将自定义的 Writing 注册到 tripod 中，
	e.SetWritings(e.WriteA) 
	// 此处需要手动将自定义的 Reading 注册到 tripod 中
	e.SetReadings(e.ReadA)

	return e
}

func main() {
	// 启用默认的配置
	startup.InitDefaultConfig()
	startup.SyncAndStartup(pow.NewPow(1024), NewExample())
}
```


#### 运行
```zsh
go build -o yu-example

./yu-example 
 
```

至此，一条区块链便启动起来了。 后续可以加入更多区块链节点来[构建区块链网络](5.5建立区块链网络.md)。
